## Proyecto: "Gestor de Gastos Personales"
Crea un programa de consola que permita gestionar tus gastos diarios. El programa debe ser capaz de hacer lo siguiente:

Registro de datos: Permitir al usuario ingresar un nombre de gasto (ej. "Almuerzo"), un monto (número) y una categoría (ej. "Comida").

Almacenamiento: Guardar estos registros en una estructura de datos (como una lista de diccionarios).

Resumen: Mostrar una función que calcule y muestre el total gastado hasta el momento.

Filtrado: Mostrar una lista de todos los gastos que pertenezcan a una categoría específica elegida por el usuario.

Persistencia (Opcional pero recomendado): Intentar guardar y cargar estos datos desde un archivo .txt o .json para que los gastos no se borren al cerrar el programa.

Reglas para este ejercicio:

No utilices bibliotecas externas (como Pandas o bases de datos). Usa solo la biblioteca estándar de Python.

Implementa el uso de, al menos, dos funciones (def).

Asegúrate de manejar errores básicos, como evitar que el programa se cierre si el usuario ingresa texto en lugar de un número al pedir el monto.
